66  /* Each node of the tree, except the scanner itself belongs to a class.
67  * The class object is just a struct declaring the owning scanner
68  * This is a pointer to the scanner as root of the tree.
69  * The type is used to identify the kind of node.
70  * The class declares pointers to a creator and a destructor method.
71  * The log and display fields are used to log and display the node.
72  * display will also display the child, sibling and parent sibling.
73  * parent, child and sibling are used to navigate the tree,
74  * from TeX box hierarchy point of view.
75  * The friend field points to a method which allows to navigate from friend to friend.
76  * A friend is a node with very close tag and line numbers.
77  * Finally, the info field point to a method giving the private node info offset.
78  */
